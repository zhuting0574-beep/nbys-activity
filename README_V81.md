# V81 逃离西撇镇对局整备与结算优化

本版本基于 V80 修复版继续修改：

1. 整备区按钮调整：
   - “保存整备”改为“保存整备以加入战局”。
   - “锁定我的整备”放在保存按钮下方并对齐。

2. 对局信息隐私规则：
   - 开局前，普通用户只能看到其他用户的呼号，看不到兵种和武器。
   - 开局后，普通用户可以看到自己小队成员的兵种和武器。
   - 管理员/超级管理员/逃离西撇镇管理员可以始终查看全部信息。

3. 终局结算物品录入优化：
   - 每个用户一行增加“物品录入”按钮。
   - 点击后弹窗录入多种获得物品和数量。
   - 终局结算时批量写入用户缓冲区仓库。
