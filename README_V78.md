# V78 - 逃离西撇镇对局管理模块

本版本基于用户上传的 `nbys-activity-dev-nbsj-20260616` 版本继续开发。

## 新增功能

1. 商店物品管理
   - 管理员功能中增加“商店物品管理”。
   - 可以编辑、删除已上架商品。

2. 对局管理
   - 逃离西撇镇导航栏新增“对局管理”。
   - 管理员可以创建 / 编辑对局：对局名称、场地、小队数、小队人数上限。
   - 用户点击对局卡片后自动加入对局。
   - 用户可选择小队、兵种等级、武器，并自行锁定。
   - 所有人锁定后，管理员可点击“开始对局”。

3. 兵种管理
   - 管理员功能新增“兵种管理”。
   - 默认兵种：跑刀仔、拖鞋军、正规军、重装兵。
   - 可编辑兵种名称、血量、每局装备维护费用。

4. 武器管理
   - 管理员功能新增“武器管理”。
   - 默认武器：刀、常规武器。
   - 特殊武器根据物品设置中“物品分类=武器”的物品自动生成。
   - 常规武器可设置使用费。
   - 特殊武器可设置每局耐久消耗百分比。

5. 开局扣费
   - 开局时自动扣除兵种装备维护费用。
   - 常规武器扣除使用费。
   - 刀不扣武器费用。
   - 特殊武器不扣现金，进入对局占用状态。
   - 跑刀仔只能选择刀。

6. 终局结算
   - 管理员可录入是否成功撤离、人头数、获得现金、获得物品。
   - 结算后现金进入用户账户，物品进入缓冲区仓库。
   - 特殊武器撤离失败会消失。
   - 成功撤离时特殊武器返回个人仓库，并扣除耐久；耐久小于等于 0 时消失。

## 数据库变更

新增表：
- `extraction_class_rules`
- `extraction_weapon_rules`
- `extraction_matches`
- `extraction_match_participants`

新增字段：
- `extraction_inventory_items.durability_percent`

程序启动时会自动创建新表和补齐字段，不需要清空数据库。
