# V86 - 逃离西撇镇仓库与重置优化

本版本基于 V85 继续修改：

1. 管理员功能弹窗去掉“商店添加新货品”入口。
   - 商店物品管理中已有新增能力，避免入口重复。

2. 重置记录改为通过“查看重置记录”按钮弹窗查看。
   - 待确认的重置申请仍会直接显示在管理员功能弹窗中。

3. 仓库默认尺寸调整。
   - 缓冲区仓库默认改为 16×48。
   - 个人初始仓库默认改为 8×12。
   - 已有小于 8×12 的个人仓库会自动补到新默认尺寸。
   - 资产重置会把个人仓库大小重置为 8×12；数据重置后新档案也使用 8×12。

4. 仓库交互优化。
   - 物品悬浮信息框提升到最上层，避免被其他物品图标遮挡。
   - 鼠标拖动物品时，会按鼠标按住物品的位置定位目标格。

5. 个人信息与赛季统计区分。
   - 个人信息中的“参加对局数”改为“总参与对局数”。
   - 个人信息新增总成功撤离率。
   - 赛季统计中的对局数改为“本赛季参与对局数”。

6. 新增“钱币”物品分类。
   - 钱币类相同物品可以在仓库中叠加。
   - 图标右下角显示数量。
   - 悬浮信息框显示数量。
   - 出售钱币时会按数量结算。

部署时继续保留：

```text
data/app.db
static/uploads/
```
