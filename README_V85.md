# V85 - 管理删除修复与逃离西撇镇重置

本版本基于 V84 继续修改：

1. 修复管理员删除用户时报 Internal Server Error 的问题。
   - 删除用户改为禁用并匿名化账号，避免历史活动、出勤、逃离西撇镇记录的外键引用导致删除失败。

2. 逃离西撇镇赛季设置增强。
   - 逃离西撇镇管理员和超级管理员可以在赛季列表中点击“编辑”修改赛季名称、起止日期和人头奖励。

3. 新增逃离西撇镇资产重置和数据重置。
   - 资产重置会清空所有用户仓库物品、甬士币并恢复默认仓库大小，保留游戏对局记录。
   - 数据重置会清空所有用户侧逃离西撇镇数据，包括资产、甬士币、用户档案、撤离记录和对局记录。
   - 两类重置都需要两名超级管理员或逃离西撇镇管理员分别输入指定确认文字。
   - 管理员功能弹窗中会显示待确认提示和最近重置记录。

4. 常规活动管理增强。
   - 非公开正式活动详情中增加“查看邀请用户”按钮，活动用户可查看额外邀请的非正式队员。
   - 用户管理列表调整列顺序：正式队员、呼号、用户名靠前，权限和邀请人移到最后两列。
   - 活动管理页新增“生效的活动/策划”，正式活动记录只显示已结束活动，“活动策划”改为“活动策划记录”并只显示已隐藏策划。
