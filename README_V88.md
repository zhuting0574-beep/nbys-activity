# V88 - 钱币拖拽堆叠与录入上限

本版本基于 V87 继续修改：

1. 钱币拖拽堆叠优化。
   - 钱币类物品拖到同类钱币图标上时，只要鼠标落在目标图标上，放开就会优先合并堆叠。
   - 这种情况下不会再自动放到附近空位。
   - 其他物品仍保持目标位置被占用时自动寻找最近可用空间。

2. 物品录入数量限制调整。
   - 普通物品单次录入上限改为 10 个。
   - 钱币类物品仍不限制总录入数量，并继续按单堆 9999 自动拆分。

3. 修复普通物品批量录入只生成 1 个的问题。
   - 非钱币物品会按最终数量创建对应数量的独立物品。
